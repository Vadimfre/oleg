# 🚴 BikeRoutes - Веломаршруты Гродно

Современная веб-платформа для велосипедистов Гродно с интерактивными маршрутами, рейтингами и комментариями.

## 🎯 Особенности

### ✨ Для пользователей
- 📍 **6 готовых веломаршрутов** с реальными GPX-файлами
- 🗺️ **Интерактивные карты** на базе Leaflet
- ⭐ **Система рейтингов** - оценивайте маршруты
- 💬 **Комментарии** - делитесь впечатлениями
- ❤️ **Избранное** - сохраняйте любимые маршруты
- 🔐 **Безопасная авторизация** через JWT + httpOnly cookies

### 🎨 Дизайн
- 🌐 **Dribbble-style** минималистичный UI
- 📱 **Полная адаптивность** под все устройства
- 🎭 **Плавные анимации** и переходы
- 🖼️ **Карусель изображений** на странице маршрута

### 🏗️ Архитектура
- ✅ **FSD (Feature-Sliced Design)** - для фронтенда
- ✅ **NestJS + Prisma** - для бэкенда
- ✅ **Next.js 15 + React 19** - современный стек
- ✅ **TypeScript** - типобезопасность

## 📦 Доступные маршруты

1. **Августовский канал** (45 км, средний)
   - Исторический маршрут вдоль канала XIX века
   - GPX: `avgust-velo.kml`

2. **Лестница в небо** (38 км, сложный)
   - Гродно - Санники через холмы
   - GPX: `Grodno_Stairway-To-Heaven_Sanniki_Grodno.gpx`

3. **Гродно - Лосево** (28 км, легкий)
   - Загородный маршрут для семейных поездок
   - GPX: `grodno-losevo.gpx`

4. **Гродно - Минск** (285 км, сложный)
   - Междугородный маршрут для профи
   - GPX: `grodno-minsk.gpx`

5. **Длинный маршрут** (95 км, сложный)
   - Тренировочный маршрут с разнообразным рельефом
   - GPX: `Long-bike.gpx`

6. **Покатушка** (15 км, легкий)
   - Вечерний маршрут по паркам Гродно
   - GPX: `pokatushka.gpx`

## 🚀 Запуск проекта

### Предварительные требования
- Node.js 18+
- npm или yarn

### Установка и запуск

#### 1. Фронтенд
```bash
npm install
npm run dev
```
Откроется на `http://localhost:3000`

#### 2. Бэкенд
```bash
cd backend
npm install
npx prisma generate
npx prisma migrate dev
npm run start:dev
```
Откроется на `http://localhost:3001`

## 🔐 Авторизация

### Требования
- **Без авторизации** можно:
  - ✅ Просматривать маршруты
  - ✅ Смотреть карты
  - ✅ Читать комментарии и рейтинги

- **С авторизацией** можно:
  - ✅ Добавлять в избранное
  - ✅ Оценивать маршруты
  - ✅ Оставлять комментарии
  - ✅ Просматривать свой профиль

### Технологии
- JWT токены в httpOnly cookies
- React Context для глобального состояния
- Protected routes

## 🗺️ GPX интеграция

Все маршруты используют реальные GPX-файлы из папки `public/gpx/`:
- Автоматический парсинг координат
- Расчет дистанции и перепада высот
- Визуализация на интерактивной карте Leaflet
- Плавные линии маршрутов

## 🎨 UI/UX особенности

### Модальные окна
- Авторизация (Login/Register)
- Оценка маршрута
- Комментарии

Все модалы:
- `z-index: 9999` для правильного отображения над картой
- Плавные анимации появления/исчезновения
- Закрытие по клику вне области

### Карты
- `z-index: 1` - не перекрывают интерфейс
- Leaflet с OpenStreetMap
- Статичный режим (только просмотр)

### Хедер
- `z-index: 9998` - всегда поверх карты
- Выпадающие меню на hover
- Профиль пользователя

## 📊 Backend API

### Auth
- `POST /auth/register` - Регистрация
- `POST /auth/login` - Вход
- `POST /auth/logout` - Выход
- `GET /auth/profile` - Профиль

### Favorites
- `POST /favorites/:routeId` - Добавить
- `DELETE /favorites/:routeId` - Удалить
- `GET /favorites` - Список
- `GET /favorites/:routeId` - Проверить

### Ratings
- `POST /ratings/:routeId` - Оценить
- `GET /ratings/:routeId/average` - Средний рейтинг
- `GET /ratings/:routeId/my` - Моя оценка
- `GET /ratings/my` - Все мои оценки

### Comments
- `POST /comments/:routeId` - Добавить
- `GET /comments/route/:routeId` - Список
- `GET /comments/my` - Мои комментарии
- `DELETE /comments/:commentId` - Удалить
